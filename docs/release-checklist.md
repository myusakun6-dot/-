# リリース前チェック（自作模試メーカー）

## 1. 機能
- [ ] 問題作成（4択/穴埋め/記述）が保存される
- [ ] 模試作成→一覧→受験開始→採点まで完走できる
- [ ] 暗記モードで回答後に答え表示できる
- [ ] 印刷（問題用/解答用）がプレビュー通りに出力される
- [ ] 共有コードの作成/取込が成功する

## 2. 課金・広告
- [ ] 24時間広告なしの付与が機能する
- [ ] 月額/年額の購入導線が表示される
- [ ] 購入復元が動く

## 3. 品質
- [ ] 主要導線が3タップ以内で到達可能
- [ ] 主要ボタンのタップ領域が44px以上
- [ ] エラー文言に対処方法が含まれている

## 4. 法務
- [ ] 利用規約リンクが開く
- [ ] プライバシーポリシーリンクが開く

## 5. 提出物
- [ ] アイコン、スクショ、説明文、サポートURLを準備
