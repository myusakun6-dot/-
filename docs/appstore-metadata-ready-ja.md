## 4. Description
自作模試メーカーは、スマートフォンで問題を作り、模試として解答し、暗記学習まで一つで完結できる学習アプリです。

主な機能:
- 問題作成（4択・穴埋め・記述）
- ジャンル管理と問題整理
- 模試の自由作成（全問題/ジャンル単位/個別選択）
- 模試受験と結果確認（問題・正答・自分の回答を一覧表示）
- 暗記モード（回答後に答え表示）
- 印刷プレビュー（問題用/解答用を分離）
- 共有コードで模試を共有・取り込み

こんな人におすすめ:
- 学校試験や資格試験の対策を自分仕様で進めたい
- 友達同士で問題を作って出し合いたい
- 本番形式の練習と暗記を1つのアプリで回したい
