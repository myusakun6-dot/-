# 自作模試メーカー テックチェックリスト

## 0. 使い方
- `[]` 未着手
- `[~]` 進行中
- `[x]` 完了

---

## 1. 機能完成
- [~] 問題作成（4択/穴埋め/記述）が正常動作
- [~] ジャンル管理（追加/絞り込み）が正常動作
- [~] 模試作成（ジャンル一括/個別選択）が正常動作
- [~] 模試受験（開始/前後移動/終了/採点）が正常動作
- [~] 暗記モード（回答後に答え表示）が正常動作
- [~] 印刷（問題用/解答用）のプレビューと出力が崩れない
- [~] 共有（コード/リンク/QR/取込）が正常動作

## 2. データ整合性
- [~] ローカル保存が再起動後も保持される
- [ ] 問題削除時に模試の参照不整合が出ない
- [ ] 模試複製/削除時にアクティブ模試が壊れない
- [ ] 共有取込時の重複問題判定が正しい
- [ ] 破損データ入力時にアプリが落ちない

## 3. 収益化（現段階）
- [ ] プラン画面で状態表示が正しい（無料/広告オフ/Pro）
- [ ] 報酬広告完了時のみ24時間広告なしが付与される
- [ ] Pro > 広告オフ > 報酬広告 > 無料 の優先順位が正しい
- [ ] 購入復元・テスト状態リセットが動作する
- [ ] 広告バナー表示条件が正しい（無料のみ表示）

## 4. UI/UX
- [ ] 主要機能へ3タップ以内で到達できる
- [ ] モバイルでタップ領域が十分（44px以上）
- [ ] エラー文言が「原因 + 対処法」になっている
- [ ] 読み込み中にボタンがローディング表示になる
- [ ] 完了時フィードバック（toast）がわかりやすい
- [ ] 画面目的が明確（作成/模試/暗記/印刷/プラン）

## 5. アクセシビリティ
- [ ] タブに `tablist/tab/tabpanel` と `aria-selected` が設定済み
- [ ] ラベル未接続の入力がない
- [ ] 色だけに依存しない状態表示（テキスト/記号）がある
- [ ] キーボード操作（左右移動/モーダルEsc）が機能する
- [ ] 文字サイズ設定が反映される（小/標準/大/システム）

## 6. パフォーマンス・安定性
- [ ] 重い操作（印刷・大量データ）で固まりにくい
- [ ] コンソールエラーが発生していない
- [ ] `node --check app.js` が通る
- [ ] Service Worker更新後に最新アセットが反映される

## 7. セキュリティ・信頼性
- [ ] `innerHTML` 使用箇所が `escapeHtml` で保護されている
- [ ] 共有コード入力の空白・改行を正規化している
- [ ] パスワード入力のバリデーション（4桁数字）が機能
- [ ] 危険操作（全削除/削除）に確認導線がある

## 8. テスト観点（最小）
- [ ] 新規ユーザー導線（データ0件）を通し確認
- [ ] 既存データ大量状態（デモ投入）で確認
- [ ] オフライン近い状態で確認（共有・印刷以外）
- [ ] iPhone Safari 最新版で確認
- [ ] Chromeでも最低限動作確認
