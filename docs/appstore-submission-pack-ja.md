# App Store 提出素材パック（自作模試メーカー）

## 1. 基本情報（入力用）
- アプリ名: 自作模試メーカー
- サブタイトル案: 問題作成・模試・暗記・印刷
- 主カテゴリ: 教育
- 副カテゴリ: 仕事効率化（任意）

## 2. 説明文（ドラフト）
自作模試メーカーは、スマートフォンで問題を作り、模試として解答し、暗記まで回せる学習アプリです。
インプットした内容を「問題作成」でアウトプットに変え、理解を定着させられます。

主な機能:
- 問題作成（4択・穴埋め・記述）
- ジャンル管理と問題整理
- 模試の自由作成（全問題/ジャンル単位/個別選択）
- 模試受験と結果確認（正答数/問題数、問題・回答・正答・判定を一覧表示）
- 暗記モード（回答後に答え表示）
- 印刷プレビュー（問題用/解答用を分離）
- 共有コードで模試を共有・取り込み

こんな人におすすめ:
- 学校試験や資格試験の対策を自分仕様で進めたい
- 友達同士で問題を作って出し合いたい
- 本番形式の練習と暗記を1つのアプリで回したい

## 3. キーワード案
模試, 試験対策, 暗記, 学習, 問題作成, 教育, 自習, 受験, 資格, テスト

## 4. プロモーションテキスト案
インプットした内容を自分で問題化してアウトプット学習。模試・暗記・印刷まで1つで完結。

## 5. スクリーンショット構成（6枚推奨）
1. ホーム: 続きから再開 + 主要導線
2. 問題作成: 4択/穴埋め/記述の作成画面
3. 模試作成: ジャンル追加と個別選択
4. 模試解答: 問題画面 + 前へ/次へ/終了
5. 暗記モード: 回答後の答え表示
6. 印刷プレビュー: 問題用/解答用切替
